### [Lv.0] [문자열 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181952) - 181952

```py
str = input()
print(str)
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = [line];
}).on('close', function () {
  str = input[0];
  console.log(str);
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        System.out.println(str);
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    cin >> str;
    cout << str;
    return 0;
}
```

각 언어별 기본 출력 방법을 물어보는 문제입니다. 난이도는 없는거나 다름없지만, 입력과 출력의 근본이 되는 문제입니다.

- Python:
  - `input()`으로 입력받은걸 `print()`로 출력합니다. 파이썬답게 직관적입니다.
  - `print(input())`도 동일한 결과를 낼 수 있지만, 문제의 요구 사항에 `str`이 언급되고 있기 때문에 굳이 생략하지는 않았습니다.
- JavaScript:
  - readline 모듈을 사용하여 `line` 이벤트 리스너를 등록하고, `close` 이벤트 리스너의 콜백 함수에 로직을 전개해서 `console.log();`로 출력합니다.
  - 파이썬대비 코드가 상당히 길지만, 가장 마지막의 console.log만 추가했습니다.
- Java:
  - Scanner 클래스를 사용하여 `next()` 메소드로 입력을 받고, `System.out.println();`으로 출력합니다.
  - 저라면 BufferedReader를 사용할 것 같지만, 이 문제에서는 굳이 고쳐쓰지는 않았습니다.
- C++:
  - string 헤더를 include하고, `cin >>`으로 입력을 받고, `cout <<`으로 출력합니다.
  - `cout << str << "\n";`도 결과는 같았습니다.

### [Lv.0] [문자열 반복해서 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181950?language=python3) - 181950

```py
str, n = input().strip().split(' ')
print(str * int(n))

```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = line.split(' ');
}).on('close', function () {
  str = input[0];
  n = Number(input[1]);
  console.log(str.repeat(n));
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        int n = sc.nextInt();
        System.out.println(str.repeat(n));
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    int n;
    cin >> str >> n;

    while(n--) {
        cout << str;
    }
    return 0;
}
```

문제 이름 그대로 각 언어별로 문자열을 반복하는 방법을 아는지 물어보는 문제입니다. 모든 언어에는 루프가 있기 때문에 그 방식으로 풀어도 됩니다.

- Python:
  - `*` 연산자를 사용해서 문자열을 반복합니다. 파이썬은 [`문자열에 숫자를 곱하면`](https://docs.python.org/3.9/library/stdtypes.html#sequence-types-list-tuple-range) 문자열이 숫자만큼 반복됩니다.
  - 대신 n이 문자열로 들어오기 때문에 연산 전에 int로 형변환을 해줘야 합니다.
- JavaScript:
  - [`String.prototype.repeat()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/repeat) 메소드를 사용해서 문자열을 반복합니다.
- Java:
  - [`String.repeat()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html#repeat(int)>) 메소드를 사용해서 문자열을 반복합니다.
- C++:
  - `while` 루프를 사용해서 n이 0이 될 때까지 문자열을 출력합니다.
  - `for` 루프를 사용해도 됩니다.

### [Lv.0] [특수문자 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181948?language=python3) - 181948

```py
print(r'!@#$%^&*(\'"<>?:;')
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

rl.on('close', function () {
  console.log(String.raw`!@#$%^&*(\'"<>?:;`);
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();

        sb.append("!@#$%^&*(")
            .append("\\")
            .append("'")
            .append("\"")
            .append("<>?:;");

        System.out.println(sb);
    }
}
```

```cpp
#include <iostream>

using namespace std;

int main(void) {
    cout << R"(!@#$%^&*(\'"<>?:;)";
    return 0;
}
```

프로그래밍 언어에서 일부 특수문자는 특별한 문법이나 의미를 가지고 있기 때문에 이스케이프 처리를 해야합니다. 이스케이프로 처리해도 되지만, 각 언어별로 raw string을 출력하는 방법을 안다면 더 간단하게 처리할 수 있습니다.

- Python:
  - 파이썬은 [`r` 접두사](https://docs.python.org/3.9/reference/lexical_analysis.html#string-and-bytes-literals)를 사용해서 raw string을 만들 수 있습니다.
- JavaScript:
  - 자바스크립트는 [`String.raw`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/raw)를 사용해서 raw string을 만들 수 있습니다.
- Java:
  - 자바 13부터 [`Text Blocks`](https://openjdk.java.net/jeps/355)라는 기능이 추가되어서 raw string을 만들 수 있습니다. 하지만 어떤 이유에서인지 플랫폼에서 막혀있는 것을 확인했기에 직접 이스케이프 처리를 했습니다.
- C++:
  - C++은 [`R` 접두사](https://en.cppreference.com/w/cpp/language/string_literal)를 사용해서 raw string을 만들 수 있습니다.

### [Lv.0] [문자열 붙여서 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181946?language=python3) - 181946

```py
str1, str2 = input().strip().split(' ')
print(str1, str2, sep="")

```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = line.split(' ');
}).on('close', function () {
  console.log(input.join(''));
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String a = sc.next();
        String b = sc.next();
        System.out.println(String.join("", a, b));
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str1, str2;
    cin >> str1 >> str2;
    cout << str1 << str2;
    return 0;
}
```

대부분의 프로그래밍 언어에서 두 개 이상의 문자열을 결합할 때 `+` 연산자를 사용하면 원하는 결과를 얻을 수 있습니다. 하지만, 각 언어별로 동일한 출력을 얻기 위한 다른 방법들이 있습니다.

- python:
  - [`print()` 함수](https://docs.python.org/3/library/functions.html#print)는 `sep` 인자를 사용해서 구분자를 변경할 수 있습니다. 기본 구분자는 공백입니다. 구분자를 `""`로 설정하면 두 문자열이 붙어서 출력됩니다.
- JavaScript:
  - 배열 메서드인 [`Array.prototype.join()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/join)를 사용하면 배열의 요소를 결합할 수 있습니다. 기본 구분자는 `,`입니다. 매개변수로 `""`를 넘기면 두 문자열이 빈틈없이 결합됩니다.
- Java:
  - 문자열 메서드인 [`String.join()`](<https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/String.html#join(java.lang.CharSequence,java.lang.CharSequence...)>) 메서드를 사용하면 문자열을 결합할 수 있습니다. 첫 번째 매개변수로 구분자를 넘기고, 두 번째 매개변수로 결합할 문자열을 넘깁니다.
- C++:
  - `cout`은 출력 버퍼이기 때문에 `<<` 연산자를 사용해서 출력할 내용을 순차적으로 넣으면 결과적으로 두 문자열이 붙어서 출력됩니다.

### [Lv.0] [문자열 돌리기](https://school.programmers.co.kr/learn/courses/30/lessons/181945?language=python3) - 181945

```py
str = input()
print("\n".join(str))
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = line;
}).on('close', function () {
  console.log(input.split('').join('\n'));
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        System.out.println(String.join("\n", str.split("")));
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    cin >> str;

    for (char ch: str)
    {
        cout << ch << "\n";
    }

    return 0;
}
```

문자열의 문자 사이사이에 개행 문자를 넣어서 새로운 문자열을 만드는 문제입니다. 기본적으로는 인덱스에 기반한 loop 또는 스트림에 기반한 loop로 해결 가능합니다. 하지만 각 언어별로 split 계통과 join 계통의 함수를 사용하면 더 간단하게 해결할 수 있습니다.

- Python:
  - 파이썬은 문자열 자체가 iterable하기 때문에 문자 단위로 따로 split 할 필요가 없습니다.
  - [`str.join()`](https://docs.python.org/3.9/library/stdtypes.html#str.join) 메서드를 사용해서 문자열 사이사이에 개행 문자를 넣을 수 있습니다.
- JavaScript:
  - 문자열 메서드 [`String.prototype.split()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split) 를 사용해서 문자열을 배열로 만듭니다.
  - 배열 메서드 [`Array.prototype.join()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/join) 를 사용해서 배열의 요소를 구분자와 결합해 문자열로 만듭니다.
- Java:
  - 문자열 메서드 [`String.split()`](<https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/String.html#split(java.lang.String,int)>)를 사용해서 문자열을 배열로 만듭니다.
  - 문자열 메서드 [`String.join()`](<https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/String.html#join(java.lang.CharSequence,java.lang.CharSequence...)>) 를 사용해서 배열의 요소를 구분자와 결합해 문자열로 만듭니다.
- C++:
  - C++에는 별도의 split 함수나 join 함수가 없습니다.
  - 범위 기반 for문을 사용해서 문자열의 각 문자를 출력합니다.
