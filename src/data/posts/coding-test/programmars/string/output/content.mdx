### [Lv.0] [문자열 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181952) - 181952

```py
str = input()
print(str)
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = [line];
}).on('close', function () {
  str = input[0];
  console.log(str);
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        System.out.println(str);
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    cin >> str;
    cout << str;
    return 0;
}
```

각 언어별 기본 출력 방법을 물어보는 문제입니다. 난이도는 없는거나 다름없지만, 입력과 출력의 근본이 되는 문제입니다.

- Python:
  - `input()`으로 입력받은걸 `print()`로 출력합니다. 파이썬답게 직관적입니다.
  - `print(input())`도 동일한 결과를 낼 수 있지만, 문제의 요구 사항에 `str`이 언급되고 있기 때문에 굳이 생략하지는 않았습니다.
- JavaScript:
  - readline 모듈을 사용하여 `line` 이벤트 리스너를 등록하고, `close` 이벤트 리스너의 콜백 함수에 로직을 전개해서 `console.log();`로 출력합니다.
  - 파이썬대비 코드가 상당히 길지만, 가장 마지막의 console.log만 추가했습니다.
- Java:
  - Scanner 클래스를 사용하여 `next()` 메소드로 입력을 받고, `System.out.println();`으로 출력합니다.
  - 저라면 BufferedReader를 사용할 것 같지만, 이 문제에서는 굳이 고쳐쓰지는 않았습니다.
- C++:
  - string 헤더를 include하고, `cin >>`으로 입력을 받고, `cout <<`으로 출력합니다.
  - `cout << str << "\n";`도 결과는 같았습니다.

### [Lv.0] [a와 b 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181951?language=python3) - 181951

```py
a, b = map(int, input().strip().split(' '))
print(f"a = {a}\nb = {b}")
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = line.split(' ');
}).on('close', function () {
  const [a, b] = input;
  console.log(`a = ${a}\nb = ${b}`);
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        System.out.printf("a = %d\nb = %d", a, b);
    }
}
```

```cpp
#include <iostream>

using namespace std;

int main(void) {
    int a;
    int b;
    cin >> a >> b;
    cout << "a = " << a << "\n" << "b = " << b;
    return 0;
}
```

포매팅 된 출력을 할 줄 아는지 물어보는 문제였습니다. 기본 출력 함수를 두 번 사용해도 되고, 개행 문자로 줄바꿈을 해도 상관없습니다.

- Python:
  - 사실상 파이썬의 표준 입력을 처리하는 가이드에 가까운 문제였습니다.
  - input을 split한 내용을 map으로 int 형변환하고, 튜플 언패킹을 통해 a, b에 할당합니다.
  - [파이썬에서 문자열 포매팅하는 방법](https://docs.python.org/3.9/library/string.html?highlight=str#custom-string-formatting)이 세가지가 있는데, 그 중에 f-string을 선호하는 편입니다.
  - `print("a = {}\nb = {}".format(a, b))`도 가능합니다.
  - `print("a = %d\nb = %d" % (a, b))`도 물론 가능합니다.
- JavaScript:
  - 배열 디스트럭처링을 통해 input 배열의 요소를 변수 a, b에 할당합니다.
  - 자바스크립트는 백틱으로 표현하는 템플릿 리터럴을 사용해서 문자열 포매팅을 쉽게 처리할 수 있습니다.
- Java:
  - 전통적인 C언어 스타일의 printf를 사용해서 포매팅을 했습니다.
  - `System.out.println(String.format("a = %d\nb = %d", a, b));`도 가능합니다.
  - 문자열이 크지 않아서 `+` 연산자로 처리해도 큰 문제는 없습니다.
- C++:
  - 가장 간단합니다. `<<` 연산자를 사용해서 문자열을 `cout` 버퍼에 넣으면 끝입니다.

### [Lv.0] [문자열 반복해서 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181950?language=python3) - 181950

```py
str, n = input().strip().split(' ')
print(str * int(n))

```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

rl.on('line', function (line) {
  input = line.split(' ');
}).on('close', function () {
  str = input[0];
  n = Number(input[1]);
  console.log(str.repeat(n));
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        int n = sc.nextInt();
        System.out.println(str.repeat(n));
    }
}
```

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    int n;
    cin >> str >> n;

    while(n--) {
        cout << str;
    }
    return 0;
}
```

문제 이름 그대로 각 언어별로 문자열을 반복하는 방법을 아는지 물어보는 문제입니다. 모든 언어에는 루프가 있기 때문에 그 방식으로 풀어도 됩니다.

- Python:
  - `*` 연산자를 사용해서 문자열을 반복합니다. 파이썬은 [`문자열에 숫자를 곱하면`](https://docs.python.org/3.9/library/stdtypes.html#sequence-types-list-tuple-range) 문자열이 숫자만큼 반복됩니다.
  - 대신 n이 문자열로 들어오기 때문에 연산 전에 int로 형변환을 해줘야 합니다.
- JavaScript:
  - [`String.prototype.repeat()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/repeat) 메소드를 사용해서 문자열을 반복합니다.
- Java:
  - [`String.repeat()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html#repeat(int)>) 메소드를 사용해서 문자열을 반복합니다.
- C++:
  - `while` 루프를 사용해서 n이 0이 될 때까지 문자열을 출력합니다.
  - `for` 루프를 사용해도 됩니다.

### [Lv.0] [대소문자 바꿔서 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181949?language=python3) - 181949

```py
str = input()
print(str.swapcase())
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

let input = [];

const swapcase = (str) => {
  return [...str]
    .map((char) => {
      if (char == char.toUpperCase()) {
        return char.toLowerCase();
      } else {
        return char.toUpperCase();
      }
    })
    .join('');
};

rl.on('line', function (line) {
  input = [line];
}).on('close', function () {
  str = input[0];
  console.log(swapcase(str));
});
```

```java
import java.util.Scanner;

public class Solution {
    static StringBuilder swapcase(String str) {
        StringBuilder sb = new StringBuilder();
        for(char c : str.toCharArray()) {
            if(Character.isUpperCase(c)) {
                sb.append(Character.toLowerCase(c));
            } else {
                sb.append(Character.toUpperCase(c));
            }
        }
        return sb;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        System.out.println(swapcase(str));
    }
}
```

```cpp
#include <iostream>
#include <string>
#include <cctype>

using namespace std;

string swapcase(const string& str) {
    string result;
    for (char c : str) {
        if (isupper(c)) {
            result += tolower(c);
        } else {
            result += toupper(c);
        }
    }
    return result;
}

int main(void) {
    string str;
    cin >> str;

    cout << swapcase(str);

    return 0;
}
```

문자열 안의 문자 처리에 대한 문제입니다. 프로그래밍 언어에 따라 Upper, Lower 관련된 함수가 다르기 때문에 ASCII 코드를 직접 이용해서 풀기도 합니다.

- Python:
  - 파이썬은 문자열 내 모든 문자의 case를 반전하는 [`str.swapcase()`](https://docs.python.org/3.9/library/stdtypes.html#str.swapcase) 메서드를 제공합니다.
  - 리스트 컴프리헨션을 활용해 `''.join(c.lower() if c.isupper() else c.upper() for c in str)`로도 풀 수 있습니다.
  - [`str.isupper()`](https://docs.python.org/3.9/library/stdtypes.html#str.isupper) 메서드를 사용해서 대문자인지 확인합니다.
  - [`str.lower()`](https://docs.python.org/3.9/library/stdtypes.html#str.lower) 메서드를 사용해서 소문자로 바꿉니다.
  - [`str.upper()`](https://docs.python.org/3.9/library/stdtypes.html#str.upper) 메서드를 사용해서 대문자로 바꿉니다.
- JavaScript:
  - [`스프레드 연산자`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Spread_syntax)를 사용해서 문자열을 문자 배열로 만듭니다. 그리고 `map()` 메소드를 사용해서 대문자면 소문자로, 소문자면 대문자로 바꿉니다.
  - 자바스크립트에는 case를 확인하는 메서드가 없기 때문에, `char == char.toUpperCase()`로 대문자인지 확인합니다.
  - [`String.prototype.toUpperCase()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/toUpperCase) 메소드를 사용해서 대문자로 바꿉니다.
  - [`String.prototype.toLowerCase()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/toLowerCase) 메소드를 사용해서 소문자로 바꿉니다.
- Java:
  - [`toCharArray()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html#toCharArray()>) 메소드를 사용해서 문자열을 문자 배열로 만듭니다. 그리고 대문자면 소문자로, 소문자면 대문자로 바꿉니다.
  - [`Character.isUpperCase()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/Character.html#isUpperCase(char)>) 메소드를 사용해서 대문자인지 확인합니다.
  - [`Character.toLowerCase()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/Character.html#toLowerCase(char)>) 메소드를 사용해서 소문자로 바꿉니다.
  - [`Character.toUpperCase()`](<https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/Character.html#toUpperCase(char)>) 메소드를 사용해서 대문자로 바꿉니다.
- C++:
  - [`cctype`](https://en.cppreference.com/w/cpp/header/cctype) 헤더에 있는 `isupper()`, `tolower()`, `toupper()` 함수를 사용해서 대문자인지 확인하고, 대문자면 소문자로, 소문자면 대문자로 바꿉니다.
  - [`const string&`](https://en.cppreference.com/w/cpp/language/reference)를 사용해서 문자열을 참조로 받아서 복사를 줄입니다.

### [Lv.0] [특수문자 출력하기](https://school.programmers.co.kr/learn/courses/30/lessons/181948?language=python3) - 181948

```py
print(r'!@#$%^&*(\'"<>?:;')
```

```js
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
});

rl.on('close', function () {
  console.log(String.raw`!@#$%^&*(\'"<>?:;`);
});
```

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();

        sb.append("!@#$%^&*(")
            .append("\\")
            .append("'")
            .append("\"")
            .append("<>?:;");

        System.out.println(sb);
    }
}
```

```cpp
#include <iostream>

using namespace std;

int main(void) {
    cout << R"(!@#$%^&*(\'"<>?:;)";
    return 0;
}
```

프로그래밍 언어에서 일부 특수문자는 특별한 문법이나 의미를 가지고 있기 때문에 이스케이프 처리를 해야합니다. 이스케이프로 처리해도 되지만, 각 언어별로 raw string을 출력하는 방법을 안다면 더 간단하게 처리할 수 있습니다.

- Python:
  - 파이썬은 [`r` 접두사](https://docs.python.org/3.9/reference/lexical_analysis.html#string-and-bytes-literals)를 사용해서 raw string을 만들 수 있습니다.
- JavaScript:
  - 자바스크립트는 [`String.raw`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/raw)를 사용해서 raw string을 만들 수 있습니다.
- Java:
  - 자바 13부터 [`Text Blocks`](https://openjdk.java.net/jeps/355)라는 기능이 추가되어서 raw string을 만들 수 있습니다. 하지만 어떤 이유에서인지 플랫폼에서 막혀있는 것을 확인했기에 직접 이스케이프 처리를 했습니다.
- C++:
  - C++은 [`R` 접두사](https://en.cppreference.com/w/cpp/language/string_literal)를 사용해서 raw string을 만들 수 있습니다.
